# Practical-6 Repository

This repository demonstrates CODEOWNERS and reusable GitHub workflows.

## Folder Structure
- frontend/: Frontend code
- backend/: Backend code
- common/: Shared modules
- .github/workflows/: GitHub Actions workflows
- .github/CODEOWNERS: Defines code ownership

## How to Use
1. Clone the repo:
   git clone https://github.com/DikshaDevi5-2005/Practical-6.git
2. Push changes to trigger workflows
3. Observe the automated CI actions
